스위프트에서는 변수나 상수를 선언할 때 반드시 숫자, 문자, 문자열, 논리같은 자료형을 지정하고 맞추어서 사용해야 한다.

## Int
양수와 음수, 0을 포함한 모든 정수 데이터를 저장하는 자료형이다.

ex) 0, 1, 2, 3, -1, -2, -3 ...

## UInt
0과 양수인 정수 데이터만 저장할 수 있는 자료형이다.

# UInt를 사용하는 이유
양의 정수 범위만 다루는 데이터를 사용할 때 메모리를 더 효율적으로 사용할 수 있다.

## Double과 Float
소수점이 포함된 실수 값을 저장해야 할 때 사용한다.

# Double
Double 자료형은 소수점 아래 15~16 자리까지 정확하게 표현할 수 있다.

# Float
Float 자료형은 소수점 아래 7~8 자리까지 정확하게 표현할 수 있다.
