문서 작업 기본 단축기

  1. 이동 커맨드
    • 화살표 : 상하좌우 한칸씩 이동
    • Ctrl + 화살표 : 한 묶음 단위로 이동
    • Home : 그 줄 맨 앞으로
    • End : 그 줄 맨 뒤로
    • PageUp : 그 페이지 맨 위로
    • PageDown : 그 페이지 맨 밑으로

  2. 선택 커맨드
    • Shift + 이동 커맨드 : 드래그
      ex) Ctrl + Shift + 화살표 : 한 묶음 드래그
    • 더블클릭 : 묶음 선택
    • Ctrl + 마우스 클릭/드래그 : 여러 대상 선택
    • Ctrl + A : 전체 선택

  3. 작업 커맨드
    • Ctrl + S : 작업 중인 파일 저장
    • Ctrl + C : 복사(원본 그대로)
    • Ctrl + X : 잘라내기(원본 지워짐, 이동시킨다고 생각)
    • Ctrl + V : 붙여넣기(복사 및 잘라내기 한 것)
    • Ctrl + Z : 작업 취소
    • Ctrl + Y : 작업 취소를 취소

    • Alt + C : 서식 복사(글자 크기/색상, 정렬 방식, 간격 등)
    • Ctrl + Alt + V : 골라 붙여넣기
      (서식 제외 텍스트만 붙여넣기 등)
      (엑셀에서 수식 제외 계산 결과만 붙여넣기할때 많이 사용)

    • Ctrl + B : 글자 굵게
    • Ctrl + U : 글자 밑줄
    • Ctrl + I : 글자 기울이기

  4. 창/탭 관련 커맨드
    • 휠클릭 : 새 탭에서 창 열기
    • Alt + Tab : 다음 창으로 전환
    • Ctrl + Tab : 다음 탭으로 전환(인터넷 창, PDF, 한글 등)
    • Alt/Ctrl + Shift + Tab : 이전 창/탭으로 전환
    • Alt + F4 : 창 닫기
    • Ctrl + W : 탭 닫기
    • Ctrl + Shift + T : 닫은 탭 복구
    • Ctrl + N : 새 탭
    • Ctrl + Shift + N : 새 시크릿 탭(크롬)
      (새 세션이기 때문에 기존 창과 로그인 연동 안됨, 여러 아이디로 로그인할 때 사용)
    • Window + 방향키 : 창 크기 및 위치 조절
      (창 잡고 화면 상하좌우 테두리로 끌어도 가능)
      (여러 창 겹치지 않게 띄워서 작업할 때 유용)
    • Window + D : 모든 창 내리기/올리기

  5. 기타 커맨드
    • PrintScreen : 화면 캡처
      (그림판, 카톡, 한글 등에 Ctrl+V로 붙여넣기)
    • Alt + PrtScr : 지금 선택된 창만 캡처
      (윈도우 키 누르고 캡처도구 검색하면, 드래그 한 부분만 캡처 가능한 프로그램도 있음)
    • Window + Shift + S : 창 캡처
      (캡처 안되는 창도 가능)

    • Insert : 글자 쓸 때 원래 그 자리에 있던 글자가 지워지면서 써지는 기능 On/Off
    • Delete : 오른쪽으로 지우기, 파일 삭제 등

    • F2 : 내용 수정(엑셀 칸, 파일 이름 등)
    • F6 : 주소창으로 커서 이동

    • Tab : (1)줄 맞춤 공백(문서) (2)다음 칸으로 이동(엑셀)
      1. 항상 같은 너비만큼 띄워지는게 아니라 문서 창에 안보이는 세로 선이 일정 간격으로 이미 그어져 있다고 생각하고 이미 입력한 텍스트로부터 거기까지만 띄워짐
        여러 줄에 텍스트를 새로 줄 맞춰서 입력하고 싶을 때 사용
      2. 화살표로 이동하는 것과는 다름
        오른쪽 화살표로 이동한 다음 엔터를 누르면 커서가 이동한 칸 밑으로 이동하지만 탭키로 이동한 다음 엔터를 누르면 처음 탭키를 눌렀던 칸 밑으로 이동함
    • Ctrl + Enter : 다음 페이지로 줄 띄우기
    • Ctrl + L/C/R : 글 왼쪽/가운데/오른쪽 정렬(한글)
    • Alt + H + M + C : 셀 병합 후 가운데 정렬(엑셀)
    • Alt + Shift + W/N : 글자 사이 간격 넓힘/좁힘
      (한글 전용 단축키, 좁은 칸 안에 글자 넣을때 사용)
    • Alt + Shift + A/Z : 줄 사이 간격 넓힘/좁힘

    • Ctrl + M + K/R/B : 글자색 검/빨/파로 변경
    • Ctrl + [/] : 글자 크기 줄이기/키우기

    • Window + E : 파일 탐색기
    • Window + R : 검색(cmd 사용 시 등)

    • Ctrl + Shift + Esc : 작업관리자
      (먹통된 창 강제로 종료할 때)
