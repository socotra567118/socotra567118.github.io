ATM에 $N$명이 기다리고 있다. 사람은 1번부터 $N$번까지 번후가 매겨져 있으며, $i$번째 사람이 돈을 인출하는데 걸리는 시간은 $P _i$이다. 이 경우, 사람들이 줄을 서는 순서에 따라 사람들이 기다리는 시간의 합이 달라지게 된다. 이 때, 사람들이 기다리는 시간의 총합의 최솟값을 구하여야 한다.

기다리는 시간의 총합의 최솟값을 구하기 위해서는 입력받은 각 사람들이 돈을 인출하는 데 걸리는 시간 배열을 오름차순으로 정렬한 후, $\sum_{k=0}^{N}(\sum_{i=0}^{k}(P _i))$를 구하여 알 수 있다.

## 입력

> 사람의 수 $N (1 \leq N \leq 1000)$<br>
> 각 사람이 돈을 인출하는 데 걸리는 시간 $P _i (1 \leq P _i \leq 1000)$

## 출력

> 사람들이 기다리는 시간의 최솟값

## Python

```python
number_of_people = int(input())
times = list(map(int, input().split()))
times.sort()

sum_of_time = 0

for i in range(number_of_people):
    sum_of_time += sum(times[0:i+1])

print(sum_of_time)
```
