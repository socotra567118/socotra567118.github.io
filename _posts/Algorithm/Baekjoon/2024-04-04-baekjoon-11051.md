자연수 $n$과 $k$가 주어졌을 때, $_n C _k$의 값을 10007로 나눈었을 때 나머지를 구하여 출력하면 된다.

| 시간 제한 | 메모리 제한 |
|:-----:|:------:|
|  1초   | 256 MB |

## Python

```python
import math

n, k = map(int, input().split())
print(math.comb(n, k) % 10007)
```
