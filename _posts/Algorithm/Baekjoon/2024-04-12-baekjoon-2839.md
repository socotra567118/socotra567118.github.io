사탕가게에 설탕을 정확히 N 킬로그램을 배달해야 한다. 설탕 봉지는 3 킬로그램 봉지와 5 킬로그램 봉지가 있다. 이때, 가장 적은 개수의 봉지로 배달하려고 할 때, 몇 개의 봉지가 필요한지 구해야 한다.

다음과 같이 알고리즘을 구현한다면 몇 개의 봉지가 필요한지 구할 수 있다.

1. N이 5로 나누어 떨어지는지 확인한다.
2. N이 5로 나누어 떨어지면 N // 5를 필요한 봉지의 수에 더하고 N이 5로 나누어 떨어지지 않으면 N에서 3을 빼고 필요한 봉지의 수에 1을 더한다.
3. 1번과 2번을 N이 음수가 될 때까지 반복한다.
4. N이 음수가 되면 정확하게 N 킬로그램을 만들 수 없는 것이므로 -1을 출력한다.

## 입력

> 첫째 줄에 N이 주어진다. (3 ≤ N ≤ 5000)

## 출력

> 최소 봉지 개수. 만약, 정확하게 N 킬로그램을 만들 수 없다면 -1

## Python

```python
N = int(input())

number_of_bags = 0

while True:
    if N % 5 == 0:
        number_of_bags += N // 5
        print(number_of_bags)
        break
    
    N -= 3
    number_of_bags += 1
    
    if N < 0:
        print(-1)
        break
```
