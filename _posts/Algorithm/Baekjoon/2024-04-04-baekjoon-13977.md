자연수 M, N, 정수 K가 주어졌을 때, M번 반복하여 $_N C _K$를 1000000007로 나눈 나머지를 출력하는 문제이다.

| 시간 제한 | 메모리 제한 |
|:-----:|:------:|
|  1초   | 512 MB |

## 주의사항

아래와 같이 Python math 라이브러리의 comb 함수를 사용하면 시간이 부족하다.

```python
import math

M = int(input())

for _ in range(M):
    N, K = map(int, input().split())
    print(math.comb(N, K) % 1000000007)
```
