입력 값을 읽고 <b>연산자</b>를 이용하여 연산을 수행한 후, 결과값을 출력하면 되는 간단한 문제이다.

## 1000번

| 시간 제한 | 메모리 제한 |
|:-----:|:------:|
|  2초   | 128 MB |

두 정수를 입력받아 두 수를 더한 후, 결과값을 출력하는 문제이다.

#### Python

```python
a, b = map(int, input().split())
print(a + b)
```

#### Swift

```swift
let numbers = readLine()!.split(separator: " ").map{ Int($0)! }
print(numbers[0] + numbers[1])
```

## 10869번

| 시간 제한 | 메모리 제한 |
|:-----:|:------:|
|  1초   | 256 MB |

두 정수를 입력받아 두 수를 각각 더하고, 빼고, 곱한 수와 두 수를 나누었을 때의 몫과 나머지를 한 줄씩 출력하는 문제이다.

```python
a, b = map(int, input().split())
print(a + b)
print(a - b)
print(a * b)
print(a // b)
print(a % b)
```
