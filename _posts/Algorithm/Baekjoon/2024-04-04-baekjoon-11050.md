자연수 n과 정수 k를 입력받아 $_n C _k$를 출력하는 문제이다. 

| 시간 제한 | 메모리 제한 |
|:-----:|:------:|
|  1초   | 256 MB |

## Python

Python의 math 라이브러리에는 $_n C _k$를 구하는 math.comb 함수가 있다.

```python
import math

n, k = map(int, input().split())
print(math.comb(n, k))
```
