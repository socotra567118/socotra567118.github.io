상의와 하의가 각각 n벌씩 있으며 i번째 옷의 색은 i이며 겹치는 색은 없다. 이때, 상의와 하의를 서로 다른 색으로 입는 경우의 수를 찾는 문제이다. 서로 다른 n개에서 순서를 고려하여 두 개를 선택하는 문제이므로 $_{n}P_{2} = n \times (n - 1)$이다.

## 입력

> N

## 출력

> 상의와 하의가 각각 다른 색이 선택되도록 하는 경우의 수

## Python

```python
n = int(input())
print(n * (n - 1))
```
