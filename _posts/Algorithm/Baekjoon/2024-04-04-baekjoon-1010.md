테스트 케이스 T번동안 강 동쪽의 M개 중에서 N개를 선택하고 선택한 N개 중에서 위쪽부터 차례대로 강 서쪽 사이트에 연결하면 되므로 $_M C _N$으로 구할 수 있다.

## 입력

> T<br>
> N M<br>
> N M<br>
> ...

## 출력

> 경우의 수 1<br>
> 경우의 수 2<br>
> 경우의 수 3<br>
> ...

## Python

```python
import math

T = int(input())

for _ in range(T):
  N, M = map(int, input().split())
  print(math.comb(M, N))
```
