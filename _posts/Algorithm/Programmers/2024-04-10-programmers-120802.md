정수 $num1$과 $num2$가 주어질 때, 두 정수의 합을 반환하는 함수를 작성하는 것이 문제이다.

## 제한사항

$-50,000 \leq num1 \leq 50,000$<br>
$-50,000 \leq num2 \leq 50,000$

## Python

```python
def solution(num1, num2):
    return num1 + num2
```

## Java

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 + num2;
    }
}
```

## Swift

```swift
import Foundation

func solution(_ num1:Int, _ num2:Int) -> Int {
    return num1 + num2
}
```
