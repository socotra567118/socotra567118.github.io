정수 $num1$과 $num2$가 주어질 때, $num1$을 $num2$로 나눈 값에 1000을 반환하는 함수를 작성해야 한다.

## 제한사항

$0 \leq num1 \leq 100$<br>
$0 \leq num2 \leq 100$

## Python

```python
import math

def solution(num1, num2):
    return math.floor(num1 / num2 * 1000)
```

## Java

```java
class Solution {
    public int solution(int num1, int num2) {
        return (int)((double)num1 / (double)num2 * 1000);
    }
}
```

## Kotlin

```kotlin
class Solution {
    fun solution(num1: Int, num2: Int): Int {
        return (num1.toDouble() / num2.toDouble() * 1000.0).toInt()
    }
}
```

## Swift

```swift
import Foundation

func solution(_ num1:Int, _ num2:Int) -> Int {
    return Int(Double(num1) / Double(num2) * 1000)
}
```
