정수 $num1$과 $num2$가 주어질 때, $num1$을 $num2$로 나눈 값에 1000을 곱한 후 정수 부분을 반환하는 함수를 작성해야 한다.

문제는 $num1$을 $num2$로 나눈 값에 1000을 곱하라고 제시되었지만, 이 순서를 따를 필요는 없다. $num1$에 1000을 곱한 후 $num2$로 나눈다면 귀찮은 형변환 과정을 없앨 수 있다. 단, **Python**에서는 **/** 를 이용하여 답을 구하면, 자동으로 형변환되어 소수점까지 반환하게 된다. 따라서, **//** 를 사용하여 몫만 반환해야 한다. 

## 제한사항

$0 \leq num1 \leq 100$<br>
$0 \leq num2 \leq 100$

## Python

```python
def solution(num1, num2):
    return num1 * 1000 // num2
```

## Java

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 * 1000 / num2;
    }
}
```

## Kotlin

```kotlin
class Solution {
    fun solution(num1: Int, num2: Int): Int {
        return num1 * 1000 / num2
    }
}
```

## Swift

```swift
import Foundation

func solution(_ num1:Int, _ num2:Int) -> Int {
    return num1 * 1000 / num2
}
```
