정수 $num1$과 $num2$가 주어질 때, $num1$을 $num2$로 나눈 몫을 반환하는 함수를 작성해야 한다.

## 제한사항

$0 \leq num1 \leq 100$<br>
$0 \leq num2 \leq 100$

## Python

```python
def solution(num1, num2):
    return num1 // num2
```

## Java

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 / num2;
    }
}
```

## Kotlin

```kotlin
class Solution {
    fun solution(num1: Int, num2: Int): Int {
        return num1 / num2
    }
}
```

## Swift

```swift
import Foundation

func solution(_ num1:Int, _ num2:Int) -> Int {
    return num1 / num2
}
```
