파이썬의 연산자에는 문자열에 사용되는 연산자와 숫자에 사용되는 연산자가 있다.

### 1. 문자열에 사용되는 연산자

##### 1) 문자열 연산자
문자열을 더하거나 반복하기 위해서 **+** 연산자와 **&#42;** 연산자를 사용할 수 있다.

```python
print("Hello," + "World")
# Hello,World

print("Hello World " * 3)
# Hello World Hello World Hello World 
```

##### 2) 문자열 인덱싱 연산자
문자열 뒤에 **대괄호 []**를 붙여 인덱싱 연산자를 사용한다. 인덱싱 연산자를 이용하여 문자열을 자를 수 있다.

### 2. 숫자에 사용되는 연산자

##### 1) 사칙연산자
숫자에 사용되는 연산자 중에서 사칙연산에 사용되는 연산자는 학교에서 수학 시간에 배운 것과 똑같이 사용할 수 있다. **+**, **-**, **&#42;**, **/**가 이에 해당한다.

```python
print(7 + 5)        # 12
print(7 - 5)        # 2
print(7 * 5)        # 35
print(7 / 5)        # 1.4
```

위 코드의 마지막 줄에서 알 수 있듯이 다른 언어와는 다르게 파이썬에서는 정수와 정수를 나누더라도 자연스럽게 실수형으로 형변환이 이루어진다.

##### 2) 기타 연산자
사칙 연산자를 제외한 다른 연산자로는 **//**, **%**, **&#42;&#42;** 등이 있다.

###### i) 몫 연산자
**//** 연산자는 두 수를 나눈 수의 몫을 반환한다.

```python
print(10 // 3)
# 3
```
<br>

###### ii) 나머지 연산자
**//** 연산자가 두 수를 나눈 수의 몫을 반환한다면, **%** 연산자는 두 수를 나눈 수의 나머지를 반환한다.

```python
print(10 % 3)
# 1
```
<br>

###### iii) 제곱 연산자
**&#42;&#42;** 연산자는 앞의 수의 두의 수 제곱을 반환한다.

```python
print(10 ** 3)
# 1000
```
앞의 수가 **10**, 뒤의 수가 **3**으므로 $10^3=1000$을 반환한다.
