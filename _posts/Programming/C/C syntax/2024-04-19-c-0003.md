* 예약어: 컴파일러와 사용이 약속된 단어
* 식별자: 필요에 따라 만들어 사용하는 단어

```c
int age;
```

에서 **int**는 예약어이고 **age**는 식별자이다.

식별자는 만들어 사용하는 단어이므로 개발자가 마음대로 사용할 수 있다. 하지만 아래 규칙은 지켜야 한다.
* 로마자 대문자 A~Z, 소문자 a~z, 숫자 0~9, 언더바 _ 로 만든다.
* 숫자로 시작할 수 없다.
* 대문자와 소문자는 서로 다르게 인식
* 예약어는 식별자로 사용할 수 없다.
