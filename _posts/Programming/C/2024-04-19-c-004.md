## 산술 연산자

수학에서도 많이 사용되는 더하기, 빼기, 곱하기, 나누기를 의미한다.

#### 더하기, 빼기, 곱하기, 음수 연산

```c
#include <stdio.h>

int main(void) {
    int a = 10;
    int b = 15;
    int add, subtract, multiply, minus;
    
    add = 10 + 15;
    subtract = 10 - 15;
    multiply = 10 * 15;
    minus = -a;
    
    printf("a: %d, b: %d\n", a, b);
    printf("덧셈: %d\n", add);
    printf("뺄셈: %d\n", substract);
    printf("곱셈: %d\n", multiply);
    printf("음수 연산: %d\n", minus);
    
    return 0;
}
```
